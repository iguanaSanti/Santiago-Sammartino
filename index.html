Escuela Taller

Aplicación web desarrollada para mostrar estados medioambientales y servicios de las playas de Gijón. Toda la información es obtenida en timpo real de los datos abiertos del Ayuntamiento de Gijón y del Puerto de Gijón. Combinando AJAX y JavaScript se obtiene los datos en formato JSON y se inyectan en el HTML.

Esta página se creó con datos abiertos de Gijón para facilitar información relativa a alojamientos, monumentos y museos a traves de llamadas de Ajax y con la Api Leaflet para tener un mapa interactivo. Además la página cuenta con una sección de tarjetas cargada automáticamente en base de la cantidad de elementos que se dispongan en los Datos Abiertos, la cual ofrece descripción y ubicación que puede ser marcada en el mapa.

En esta página se basa en el mismo funcionamiento que la de Puntos de Interés. Mediante llamadas de Ajax se obtienen de los JSON del Ayuntamiento de Gijón. Aquí se plasman todos los datos, a través de un gráfico SVG, relativos de las tarjetas ciudadanas registradas desde el 2001 hasta el último año, pudiendo selecionar el año y diferenciando las tarjetas físicas y jurídicas.

Este sitio fue el primer proyecto que se elaboró en el proyecto formartivo Escuela Taller en el cual se desarrollaron las primeras habilidades de programación y coordinación entre equipos utilizando principalmente HTML, JavaScript y CSS. En el sitio se crearon menús desplegables, carruseles, calendarios y bloques de noticias entre otros, todo ello relacionado con el proyecto, dotándolo de visibilidad como muestra de los trabajos que se llevan a cabo.
